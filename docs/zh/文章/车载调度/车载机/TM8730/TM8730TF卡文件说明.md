# TM8730 TF卡文件说明

```
TF卡
 ├─ bzqmp3 ----------------- 语音文件
 |   ├─ B1 ----------------- 语音线路
 |   ├─ ... ---------------- n条语音线路
 |   └─ B2 ----------------- 语音线路
 └─ SYS -------------------- 系统使用的资源文件
     ├─ FONT --------------- 字体
     ├─ MCU
     ├─ PIC ---------------- 图片
     └─ TSY ---------------- 提示音语音文件
```
